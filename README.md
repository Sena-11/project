# Project SENA
Para inicializar el servidor en local para su desarrollo:
 - Clonar el repo.
        ```  
            git clone https://github.com/Sena-11/project.git
        ```
 - Crear documento .env con:
    ```env
        PORT=3000
    ```
 - Instalar dependencias
        ```
            npm install
        ```
 - Correr en modo dev
        ```
            npm run dev
        ```
